# Olá, eu sou a Tainara 👋

Bibliotecária formada pela **Universidade de Brasília (UnB)**, com atuação voltada para **Data Analytics**, **Ciência de Dados** e **Inteligência Artificial**, aplicando análise de dados para transformar informações em insights que apoiam a tomada de decisão.

Neste perfil compartilho projetos desenvolvidos com **Python, SQL, Power BI, Machine Learning, APIs, Engenharia de Dados e Visualização de Dados**, além de estudos sobre **Business Intelligence, ETL, automação de processos e análise exploratória de dados (EDA)**.

Minha experiência em **Gestão da Informação**, organização e qualidade de dados, aliada à formação em Ciência de Dados, permite conectar conhecimento de negócio, governança de dados e desenvolvimento de soluções analíticas orientadas por resultados.

* **Visualização:** Power BI, Plotly, Matplotlib
* **Engenharia de Dados:** ETL, APIs, Apache Airflow
